# mpbirthday
Code for the BlueSky bot [@mpbirthday](https://bsky.app/profile/mpbirthday.bsky.social)

This bot congratulates current and former members of the Bundestag (except those from the AfD) on their birthdays. It is based on a parsed version of the dataset "[Stammdaten aller Abgeordneten seit 1949 im XML-Format](https://www.bundestag.de/resource/blob/472878/MdB-Stammdaten.zip) as provided by the [Open Data Service](https://www.bundestag.de/services/opendata) of the German Bundestag. Additionally, the messages are enriched with a Wikipedia link as retrieved from [Wikidata](https://www.wikidata.org/wiki/Wikidata:Main_Page).
